# DailyChallenge

## 136只出现一次的数字

20200514

### 题目描述

给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

说明：

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

### 解答

位运算，异或的理解。

- 任何数和0做异或运算，结果仍然是原来的数
- 任何数和1做异或运算，结果取非
- 任何数和其自身做异或运算，结果是0
- 异或运算满足交换律和结合律

```java
class Solution {
    public int singleNumber(int[] nums) {
        int res = 0;
        for(int num : nums){
            res ^= num;
        }
        return res;

    }
}
```
